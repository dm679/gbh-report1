# Маппинг UF-полей — globalbalihome.bitrix24.eu (РЕШЕНО)

Подписи полей Битрикс через REST отдаёт в `formLabel`/`listLabel` (метод
`crm.deal.fields`), значения списков — там же в `items`. По ним определён маппинг.

## Итоговый маппинг (прописан в `js/config.js`)

| Понятие в отчёте | UF-поле | Тип | Примечание |
|---|---|---|---|
| **Проект** | `UF_CRM_6993017AC1684` («Project») | строка | = название папки товара / верхний уровень продукта (напр. «Serenity Village») |
| **Агентство** | `UF_CRM_6986145D2F3DD` («Агенство») | crm → **Компания** | ID компании резолвится в название (`crm.company.list`) |
| **Агент** | `UF_CRM_6986145D3987E` («Агент») | crm → **Контакт** | ID контакта резолвится в ФИО (`crm.contact.list`) |
| **Направление продаж** | = **Воронка** (`CATEGORY_ID`) | — | по решению заказчика; отдельное UF не используется |

Воронки портала: `0` RED, `1` B2B leads, `3` В2В company, `5` SOUL AI QB.

## Заметки / открытые моменты

1. **«Направление продаж»** — на портале существует и UF-поле с таким названием
   (`UF_CRM_1770370062`, значения RU/UA/ENG), но по решению заказчика направление =
   воронка. Если понадобится именно это поле — добавить его в `CONFIG.uf` и в разрезы.
2. **Проект** — это строковое поле (= название папки товара, верхний уровень
   продукта), не справочник. Значения берутся как есть; список для фильтра
   собирается из встретившихся в сделках значений.
3. **Часть данных может быть не заполнена** (портал только запускается) — это
   штатно: пустые значения попадают в «Без проекта / Без агентства / Без агента»,
   пустой `SOURCE_ID` — в «—», нулевой `OPPORTUNITY` не ломает расчёт.
4. **Сумма сделки** берётся из стандартного поля `OPPORTUNITY` (подтверждено: в
   воронке 5 заполнено; в воронке 0 «RED» суммы ещё не проставлены).
5. **Валюта**: базовая валюта портала — USD; курс берётся из
   `AMOUNT`/`AMOUNT_CNT` (`crm.currency.list`). Отчёт — только в USD.

## Если поля изменятся / для другого портала

Поменяйте коды в `js/config.js` → `CONFIG.uf` (и `CONFIG.ufBind` для crm-привязок).
Получить список полей с названиями:
```js
BX.callMethod('crm.deal.fields').then(r => {
  Object.entries(r.items).filter(([k]) => k.startsWith('UF_CRM_'))
    .forEach(([k, f]) => console.log(k, f.type, f.formLabel));
});
```
